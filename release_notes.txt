# Cortex Multimodal Proxy v3.2.0 — Release Notes

## What's new

Cortex Multimodal Proxy v3.2.0 introduces a runtime provider switch so the same proxy can serve your OpenCode Go subscription, your DeepSeek account, or both — without changing the OpenCode TUI configuration.

Pick your mode with a single environment variable:

- **`auto`** — recommended default. Uses your OpenCode Go brains if that key is set, otherwise your DeepSeek account.
- **`opencode`** — exclusively the four OpenCode Go text brains plus MiMo V2.5 image vision.
- **`deepseek`** — exclusively DeepSeek V4 Pro/Flash under your account, plus MiniMax M3 as the native multimodal passthrough (image and video in one shot).
- **`hybrid`** — both providers loaded. OpenCode Go brains keep their standard names; your DeepSeek account shows up under `proxy/local-deepseek-v4-pro` and `proxy/local-deepseek-v4-flash`.

Existing clients do not need to change. The `auto` mode preserves the v3.1.0 wire format exactly: the same four brains, the same `mimo-v2.5` passthrough, the same `/v1/models` listing, the same SSE streaming behaviour.

## Why this matters

- **One proxy, two ecosystems** — no more maintaining separate installations for OpenCode Go and direct DeepSeek.
- **Cost-aware routing** — when only the DeepSeek key is set, your OpenCode Go brains are dropped (no idle $10/month dependency) and DeepSeek's post-June 2026 price cut kicks in: $0.435 input / $0.87 output per million tokens.
- **Native video support** — DeepSeek mode pairs with MiniMax M3, which handles both image and video in one upstream call (no separate audio/video/PDF fallback chain needed for those media types).
- **Two ready-to-use OpenCode TUI templates** committed in the repo (`opencode.json` for OpenCode Go flavor, `opencode.deepseek.json` for DeepSeek flavor); `scripts/select-opencode-config.sh` is a printed reminder, never a silent rewrite.

## What's stable

- All four OpenCode Go brains — `proxy/glm-5.2`, `proxy/deepseek-v4-pro`, `proxy/qwen3.7-max`, `proxy/mimo-v2.5-pro` — unchanged in API surface and pricing.
- MiMo V2.5 senses layer (image description, $0.14 / $0.28 per 1M) — unchanged.
- Gemini 2.5 Flash fallback for audio/PDF — unchanged.
- Claude Code compatibility (`/v1/messages` with Anthropic-format) — unchanged.
- `/health`, `/v1/models`, SSE streaming, retry policy — unchanged.

## Upgrading

1. Pull the new image / checkout the tag.
2. Pick a mode by adding `BRAIN_MODE=<auto|opencode|deepseek|hybrid>` to your `.env`.
3. If you want the new DeepSeek or MiniMax M3 capabilities, add `DEEPSEEK_API_KEY` and/or `MINIMAX_API_KEY` to `.env` too.
4. Restart the container. No client reconfiguration required.
